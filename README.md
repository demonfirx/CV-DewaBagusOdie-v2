## 🚀 Overview

This is the official portfolio website for **Dewa Bagus Odie Ocxivian**. Designed with a premium aesthetic, featuring sharp typography, fluid animations, and a structured layout that reflects professional excellence in frontend design and development.

Original design concept: [Figma Design](https://www.figma.com/design/OEXIdHteUXboFgfA7Xw9ik/Build-ODIE-Portfolio-Website)

## ✨ Key Features

- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop.
- 🌓 **Adaptive Theme** – Intelligent Light/Dark mode transitions.
- 🎨 **Premium Visuals** – Custom orange diagonal background pattern for depth.
- ⚡ **High Performance** – Built with Vite 6.x and Tailwind 4.x for blazing fast loads.
- 🎬 **Smooth Animations** – Powered by Motion for a curated user experience.
- 📞 **One-Tap Contact** – Direct integration with WhatsApp and Email.

## 🛠️ Tech Stack

- **Framework:** React 18
- **Bundler:** Vite 6
- **Styling:** Tailwind CSS 4 (Custom Utilities)
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
- **Components:** Radix UI

## 🏁 Getting Started

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 2. Installation
```bash
npm install
```

### 3. Development Server
```bash
npm run dev
```
